# videoPlus 사용 가이드

## 1. 설치하기

### macOS

1. 다운로드 폴더에 videoPlus.zip을 받아 압축을 풀어주세요.
2. Spotlight를 실행하고 **터미널**을 열어주세요.
3. 터미널에
   `chmod +x ~/Downloads/videoPlus/install-mac.sh && ~/Downloads/videoPlus/install-mac.sh`
   입력 후 엔터를 눌러주세요.

> 만약 터미널 창에 ⚠️ "Operation not permitted"라는 오류가 발생하면 아래
> 명령어를 대신 입력해주세요.
> `xattr -cr ~/Downloads/videoPlus && chmod +x ~/Downloads/videoPlus/install-mac.sh && ~/Downloads/videoPlus/install-mac.sh` 5.
> 설치 완료를 확인합니다. 6. Premiere Pro를 재시작한 후 **창(Window) >
> 확장(Extensions) > videoPlus**를 선택합니다.

> [스크린샷: 터미널 설치 화면]

### Windows

1. 다운로드 폴더에 videoPlus.zip을 받아 압축을 풀어주세요.
2. `install-windows.bat` 파일을 더블클릭합니다.
3. 설치 완료를 확인합니다.
4. Premiere Pro를 재시작한 후 **창(Window) > 확장(Extensions) > videoPlus**를
   선택합니다.

> [스크린샷: Windows 설치 화면]

---

## 2. 시작하기

### 패널 열기

Premiere Pro 상단 메뉴에서 **창(Window) > 확장(Extensions) > videoPlus**를
클릭하면 패널이 열립니다.

> [스크린샷: Extensions 메뉴에서 videoPlus 선택]

### 시퀀스 연결 확인

패널 우측 상단에 초록색 점과 시퀀스 이름이 표시되면 정상 연결된 상태입니다.

> [스크린샷: 헤더 영역 - 연결됨 · 시퀀스명]

### 시퀀스 새로고침

Premiere에서 다른 시퀀스로 전환했거나 연결이 끊긴 경우, 헤더 우측 끝의 🔄
**새로고침** 버튼을 클릭하여 현재 시퀀스 정보를 다시 불러옵니다.

- 같은 시퀀스에서 누르면 데이터는 그대로 유지됩니다.
- 다른 시퀀스로 전환 후 누르면, 해당 시퀀스에 저장된 작업이 있을 경우 상단에
  **불러오기 배너**가 표시됩니다.

> [스크린샷: 헤더 새로고침 버튼]

## 3. 받아쓰기

타임라인의 오디오를 자동으로 텍스트로 변환합니다.

### 3-1. 받아쓰기 준비 (트랙 / 화자 선택)

받아쓰기를 시작하기 전 초기 화면에서 두 가지를 설정합니다.

**받아쓸 오디오 트랙**

시퀀스에서 클립이 있는 오디오 트랙들이 체크박스로 표시됩니다. 받아쓰기에 사용할
트랙을 골라 체크하세요.

- 트랙 이름과 클립 개수가 함께 표시됩니다.
- 하나도 선택하지 않으면 **받아쓰기 시작** 버튼이 비활성화되고 "오디오를
  선택해주세요" 안내가 나타납니다.
- 인터뷰처럼 화자별로 트랙이 분리된 영상은 해당 트랙들을 모두 선택하세요. 여러
  트랙을 함께 선택하면 트랙 간 음성 누설(크로스토크)을 자동으로 제거한 뒤
  받아쓰기에 사용하므로 화자 분리 정확도가 올라갑니다.

**화자 수**

인터뷰 영상이라면 2명, 혼자 말하는 영상이라면 1명을 선택하세요.

> [스크린샷: 초기 화면 - 트랙 선택 + 화자 수 + 받아쓰기 시작 버튼]

### 3-2. 받아쓰기 시작

**받아쓰기 시작** 버튼을 누르면 오디오 추출 → 서버 업로드 → 텍스트 변환이
자동으로 진행됩니다.

- 1시간 짜리 영상 기준 평균 4~5분 소요됩니다.
- 진행 중에는 아이콘과 텍스트가 변경되며, 취소 버튼으로 중단할 수 있습니다.

> [스크린샷: 받아쓰기 진행 중 화면]

### 3-3. 받아쓰기 완료

완료되면 컷편집 화면으로 자동 전환됩니다. 왼쪽 사이드바에 요약이, 오른쪽에 문장
리스트가 표시됩니다.

> [스크린샷: 받아쓰기 완료 후 컷편집 화면]

### 3-4. 다시 받아쓰기

받아쓰기 결과를 새로 만들고 싶을 때는 컷편집 화면 왼쪽 사이드바의 **다시
받아쓰기** 버튼을 누릅니다.

- 버튼을 누르면 초기 화면(트랙/화자 선택 화면)으로 돌아갑니다.
- 트랙과 화자 수를 다시 지정한 뒤 **받아쓰기 시작**을 누르면 새 받아쓰기가
  진행됩니다.
- 이전 받아쓰기 결과는 헤더의 시계 아이콘(히스토리)에서 복원할 수 있습니다.

> [스크린샷: 다시 받아쓰기 버튼]

### 3-5. 이전 작업 불러오기

이전에 받아쓴 기록이 있으면 상단에 불러오기 배너가 표시됩니다. **불러오기**
버튼을 눌러 이전 작업을 복원할 수 있습니다.

> [스크린샷: 불러오기 배너]

## 4. 컷편집

불필요한 무음, 간투사를 찾아 삭제하여 영상을 정리합니다.

### 4-1. 화면 구성

| 영역          | 설명                                     |
| ------------- | ---------------------------------------- |
| 왼쪽 사이드바 | 최소 무음 길이 조절, 다시 받아쓰기, 요약 |
| 오른쪽 상단   | 무음 선택, 간투사 선택, 시퀀스 적용 버튼 |
| 오른쪽 중앙   | 문장 리스트 (단어별 표시)                |
| 오른쪽 하단   | 오디오 파형                              |

> [스크린샷: 컷편집 전체 화면 구성]

### 4-2. 무음/간투사 선택

상단의 **무음 선택** 또는 **간투사 선택** 버튼을 클릭하면 해당 단어가 빨간색
취소선으로 표시됩니다.

- 이미 삭제된 단어: 회색 취소선
- 삭제 예정(선택된) 단어: 빨간색 취소선

> [스크린샷: 무음/간투사 선택된 상태]

### 4-3. 단어 개별 선택

문장 리스트에서 단어를 클릭하여 포커스한 후 **K키**를 누르면 개별 단어를
선택/해제할 수 있습니다.

### 4-4. 시퀀스 적용

선택이 완료되면 우측 상단의 **시퀀스 적용** 버튼을 클릭합니다. 선택된 단어 수가
괄호 안에 표시됩니다.

> [스크린샷: 시퀀스 적용 버튼 (선택 수 표시)]

### 4-5. 최소 무음 길이 조절

왼쪽 사이드바의 슬라이더로 최소 무음 길이를 조절합니다. 기준 이하의 짧은 무음은
화면에서 숨겨집니다.

> [스크린샷: 최소 무음 길이 슬라이더]

### 4-6. 파형 활용

하단 파형에서 현재 재생 위치를 확인하고, 클릭하여 해당 위치로 이동할 수
있습니다. 각 단어는 노란색 영역(region)으로 표시됩니다.

**단어 경계 조정**

각 단어 영역의 좌/우 핸들을 끌어 단어의 시작·끝 시간을 직접 조정할 수 있습니다.

- 양옆 단어와 겹치지 않는 범위 안에서만 이동합니다. 한계에 닿으면 핸들이 잠깐
  빨갛게 깜박입니다.
- 조정한 단어를 컷 적용하면 새 경계 기준으로 잘립니다. 단어 앞뒤의 여백이나
  숨소리를 함께 잘라낼 때 유용합니다.

> [스크린샷: 파형 패널 — 단어 경계 드래그]

**원본 시간으로 되돌리기 (↺)**

드래그로 단어 시작/끝 시간이 받아쓰기 당시와 달라진 region은 **노란 테두리**로
표시되고 우측 하단에 **↺ 버튼**이 나타납니다.

- ↺ 버튼을 누르면 해당 단어가 받아쓰기 직후 원본 시간으로 즉시 복원됩니다.
- 복원되면 노란 테두리와 ↺ 버튼도 함께 사라집니다.

> [스크린샷: 드래그된 region의 노란 테두리 + ↺ 버튼]

**겹친 region 관리 (▽/△)**

화자가 2명 이상일 때 각 region 우측 상단에 **▽ 버튼**이 표시됩니다. 두 화자의
파형 region이 겹쳐 있어서 뒤쪽 region의 드래그 핸들이 잘 잡히지 않을 때
사용합니다.

- **▽**: 클릭하면 해당 region이 반투명해지면서 마우스 이벤트가 통과하여 뒤쪽
  region의 핸들을 조작할 수 있습니다. 버튼은 **△**로 바뀝니다.
- **△**: 다시 클릭하면 원래 불투명 상태로 복원됩니다.
- 문장 리스트에서 단어를 클릭하면 해당 region이 자동으로 맨 앞 + 불투명
  상태로 복원됩니다.

> [스크린샷: 겹친 region — ▽ 토글 후 뒤 region 드래그]

### 4-7. 요약 사이드바

왼쪽 사이드바에 영상 내용 요약이 표시됩니다. 시간 뱃지를 클릭하면 해당 위치로
바로 이동합니다.

요약 생성에 실패한 경우 ⚠ 경고 아이콘과 함께 **다시 쓰기** 버튼이 나타납니다. 이
버튼을 누르면 같은 받아쓰기 결과로 요약을 다시 요청합니다.

> [스크린샷: 요약 사이드바 — 다시 쓰기 버튼]

**요약 안에서 단어 찾기 (🔍)**

요약 헤더의 🔍 아이콘을 클릭하면 요약 영역 상단에 작은 검색 바가 나타납니다.

- 검색어 입력 → 매치된 글자에 노란 하이라이트, 자동으로 첫 매치로 스크롤
- **Enter** 다음 매치, **Shift+Enter** 이전 매치
- **Aa** 대소문자 구분 토글
- **Esc** 또는 ✕ 닫기

> [스크린샷: 요약 검색 바 + 하이라이트]

### 4-8. 화자 변경

각 문장 오른쪽의 체크박스(☐)를 선택하면 상단에 화자 변경 UI가 나타납니다.

- **개별 선택**: 체크박스 클릭
- **같은 화자 모두 선택**: 체크박스 옆 ▼ 화살표 클릭 → "화자 A 모두 선택"
- 선택 후 원하는 화자 버튼을 클릭하면 일괄 변경됩니다.
- 새 화자 추가 또는 화자 이름 지정은 헤더의 👥 **화자 관리** 버튼을 사용하세요
  (6. 화자 관리 참고).

> [스크린샷: 화자 일괄 변경 UI]

### 4-9. 히스토리

헤더의 시계 아이콘을 클릭하면 백업 히스토리를 확인하고, 이전 상태로 복원할 수
있습니다.

> [스크린샷: 히스토리 다이얼로그]

### 4-10. 단어 찾기 / 바꾸기

문장 안의 단어를 검색하고 일괄 변경할 수 있습니다. 컷편집/자막편집 두 탭 모두
동일하게 동작합니다.

**열기**

- 탭 헤더 우측의 🔍 버튼 클릭
- 또는 **Cmd/Ctrl + F** 단축키
- 우측에 검색 사이드바가 펼쳐집니다. 같은 방식으로 다시 누르면 닫힙니다.

> [스크린샷: 검색 사이드바 열린 상태]

**찾기**

- 검색어 입력 → 매치된 글자에 노란 하이라이트, 현재 매치는 진한 노랑 + 테두리
- **Enter** 다음 매치, **Shift+Enter** 이전 매치
- 사이드바 아래쪽 매치 리스트에서 항목 클릭 시 본문이 해당 단어로 점프
- **Aa** 대소문자 구분, **ab** 단어 단위 매치 토글

> 검색은 단어 단위로만 동작합니다. 띄어쓰기 포함된 구문(예: `the man`)은 검색
> 결과가 없습니다.

> [스크린샷: 검색어 입력 + 매치 리스트]

**바꾸기**

검색 사이드바의 **[+ 바꾸기]** 토글을 클릭하면 치환 영역이 펼쳐지면서 리스트의
각 매치 옆에 체크박스가 나타납니다.

- 바꿀 단어 입력 → 매치 리스트에 미리보기 표시 (~~원본~~ + 새 단어)
- **바꾸기**: 현재 매치 한 개만 변경
- **선택 바꾸기 (N)**: 체크된 매치를 모두 변경 (기본은 모두 체크됨)
- 체크박스 해제로 일부 제외 가능, 상단 "모두 선택"으로 전체 토글
- 4개 이상 변경 시 확인 다이얼로그가 표시됩니다.

> [스크린샷: 바꾸기 UI — 미리보기 + 체크박스]

**탭별 독립 동작**

- 컷편집의 검색/바꾸기는 컷편집 단어에만 적용됩니다.
- 자막편집의 검색/바꾸기는 자막편집 단어에만 적용됩니다.
- 두 탭은 검색 상태가 독립적으로 유지됩니다.

## 5. 자막편집

받아쓴 텍스트를 자막용으로 편집하고 시퀀스에 캡션으로 적용합니다.

### 5-1. 탭 전환

상단의 **자막편집** 탭을 클릭하여 자막편집 모드로 전환합니다.

> [스크린샷: 탭 메뉴 - 자막편집 선택]

### 5-2. 줄 당 단어 수 조절

상단 슬라이더로 한 자막에 표시할 최대 단어 수를 조절합니다 (2~8단어). 단어 수에
따라 문장이 자동으로 나뉩니다.

> [스크린샷: 단어 수 슬라이더]

### 5-3. 단어 수정

수정하고 싶은 단어를 **더블클릭**하면 편집 모드가 됩니다.

- **Enter**: 수정 확인
- **Esc**: 수정 취소
- 다른 곳 클릭: 수정 확인

> [스크린샷: 단어 편집 모드]

### 5-4. 문장 나누기 / 합치기

| 키            | 기능                                           |
| ------------- | ---------------------------------------------- |
| **Enter**     | 현재 포커스된 단어 위치에서 문장을 나눕니다    |
| **Backspace** | 현재 문장을 이전 문장과 합칩니다               |
| **N**         | 현재 단어를 이전 단어와 합칩니다 (텍스트 병합) |
| **M**         | 현재 단어를 다음 단어와 합칩니다 (텍스트 병합) |

> [스크린샷: 문장 나누기/합치기 전후 비교]

### 5-5. 실행 취소 / 다시 실행

헤더의 ↩ / ↪ 버튼 또는 **Ctrl+Z** / **Ctrl+Shift+Z**로 자막편집 작업을
되돌리거나 다시 실행할 수 있습니다.

### 5-6. 화자 변경

컷편집과 동일한 방식으로 체크박스를 사용하여 화자를 변경할 수 있습니다.

- 체크박스 옆 ▼ 클릭 → "화자 A 모두 선택"
- 상단에서 변경할 화자 선택
- 새 화자 추가는 헤더의 👥 **화자 관리** 버튼에서 가능 (6. 화자 관리 참고)

> [스크린샷: 자막편집 화자 변경]

### 5-7. 시퀀스에 자막 적용

편집이 완료되면 우측 상단의 **시퀀스에 자막 적용** 버튼을 클릭합니다. 화자별로
별도의 캡션 트랙이 생성됩니다.

> [스크린샷: 캡션 적용 후 시퀀스 타임라인]

### 5-8. 단어 찾기 / 바꾸기

자막편집 탭에서도 검색·치환을 사용할 수 있습니다. 자세한 사용법은 [4-10. 단어
찾기 / 바꾸기](#4-10-단어-찾기--바꾸기) 참고. 컷편집과 별도로 동작하며 자막
단어 텍스트에만 영향을 줍니다.

## 6. 화자 관리

화자별 이름을 지정하거나 화자를 추가/삭제할 수 있습니다. 헤더 우측의 👥 **화자
이름 지정** 아이콘을 클릭하면 다이얼로그가 열립니다.

> [스크린샷: 화자 관리 헤더 버튼]

### 6-1. 화자 이름 지정

각 화자 옆의 입력란에 이름을 입력하고 **저장** 버튼을 클릭합니다.

- 이름이 지정되면 문장 왼쪽의 화자 표시 (`A`, `B` 등)가 입력한 이름으로
  바뀝니다.
- 화자 변경 버튼, 자막 다운로드 등 모든 화자 표시에 반영됩니다.
- 저장 시 현재 편집 상태와 함께 자동으로 서버에 저장됩니다.

> [스크린샷: 화자 이름 지정 다이얼로그]

### 6-2. 화자 추가

**+ 화자 추가** 버튼을 클릭하면 새 화자가 추가됩니다 (최대 6명).

- 추가된 화자는 문장 선택 시 변경 가능한 옵션으로 나타납니다.
- 이름 없이 추가하면 기본 라벨(A, B, C, D, E, F)로 표시됩니다.

### 6-3. 화자 삭제

화자 항목 오른쪽의 🗑 아이콘으로 삭제합니다.

- **사용 중인 화자**(해당 화자의 문장이 하나라도 있는 경우)는 삭제할 수
  없습니다. 마우스를 올리면 안내 툴팁이 표시됩니다.
- 최소 1명은 남아있어야 하므로, 마지막 화자는 삭제할 수 없습니다.

## 7. 다운로드

자막 텍스트, 요약, SRT 파일을 PC에 저장합니다. 헤더 우측의 ⬇ **다운로드**
아이콘을 클릭하면 다이얼로그가 열립니다.

> [스크린샷: 다운로드 헤더 버튼]

### 7-1. 파일 유형 선택

세 가지 파일 유형을 동시에 선택할 수 있으며, 선택한 파일들이 순차적으로
저장됩니다.

| 유형                  | 설명                               |
| --------------------- | ---------------------------------- |
| 일반 자막 텍스트 파일 | 자막을 일반 텍스트(.txt)로 저장    |
| 요약 텍스트 파일      | 영상 요약을 텍스트(.txt)로 저장    |
| SRT 파일              | 자막을 표준 SRT 자막 형식으로 저장 |

> [스크린샷: 다운로드 다이얼로그 - 파일 유형 선택]

### 7-2. 자막 텍스트 옵션

**일반 자막 텍스트 파일**을 선택하면 아래 옵션이 나타납니다:

- **타임코드**: 각 줄 앞에 시간 정보(`[00:00:03]`)를 포함합니다.
- **화자 이름**: 각 줄 앞에 화자 이름(또는 A/B/C)을 포함합니다.

> [스크린샷: 자막 텍스트 옵션]

### 7-3. SRT 옵션

**SRT 파일**을 선택하면 두 가지 모드 중 하나를 선택합니다:

- **화자별** (기본): 화자 수만큼 별도 SRT 파일이 생성됩니다. 파일명은
  `subtitle_{화자이름}_{timestamp}.srt` 형식입니다.
- **통합**: 모든 화자가 하나의 SRT 파일로 저장됩니다.

> [스크린샷: SRT 모드 선택]

### 7-4. 파일 위치

기본 저장 위치는 **바탕화면**입니다. **변경** 버튼으로 다른 폴더를 지정할 수
있습니다.

> [스크린샷: 파일 위치 선택]

### 7-5. 다운로드 실행

**다운로드** 버튼을 클릭하면 파일이 저장되고, 잠시 후 "저장 완료" 표시와 함께
다이얼로그가 자동으로 닫힙니다.

- 파일 유형을 하나도 선택하지 않으면 경고 메시지가 표시됩니다.
- 요약본이 없는 경우 "요약 텍스트 파일" 옵션은 비활성화됩니다.

> [스크린샷: 저장 완료 화면]

## 9. 데이터 저장 위치

작업 중 만들어지는 모든 시퀀스는 프로젝트 패널의 **videoPlus Backups** 빈에
자동으로 보관됩니다. 빈 내부는 두 영역으로 나뉩니다.

| 하위 폴더         | 보관 내용                                                                    |
| ----------------- | ---------------------------------------------------------------------------- |
| **original**      | 받아쓰기 직전의 **원본 시퀀스** (받아쓰기 시작 시 1회만 생성)                |
| **`<시퀀스 ID>`** | 편집 작업 직전의 **스냅샷 히스토리** (컷 일괄 적용 등 매 편집 직전마다 추가) |

### 원본으로 되돌아가기

- **받아쓰기 시작** 버튼을 누르는 시점에 현재 시퀀스가 자동으로 복제되고, 이후
  편집은 항상 복제본(작업용 새 시퀀스)에서 진행됩니다.
- 원본을 다시 보려면 **videoPlus Backups > original** 폴더에서 해당 시퀀스(이름
  끝에 `_<날짜시간>` 표기)를 더블클릭하세요.

### 편집 히스토리 복원

- 헤더의 🕘 시계 아이콘에서 백업 목록을 확인하고 원하는 시점으로 복원할 수
  있습니다. 백업 파일을 직접 열 필요 없이 다이얼로그에서 처리됩니다.
- 복원하면 직전까지 작업 중이던 시퀀스는 자동으로 **videoPlus Archive** 빈으로
  이동되어 안전하게 보관됩니다.

> [스크린샷: 프로젝트 패널 - videoPlus Backups 구조]

### 캡션 파일

- 자막 적용 시 SRT 파일이 프로젝트 파일(.prproj)과 같은 폴더의 **captions**
  폴더에 저장됩니다.
- 화자별로 별도 SRT 파일이 생성됩니다 (예: `speakerA_xxxxx.srt`,
  `speakerB_xxxxx.srt`).
- Premiere 프로젝트 패널의 **videoPlus Captions** 빈에 자동 임포트되어 캡션
  트랙으로 추가됩니다.

> [스크린샷: 프로젝트 패널 - videoPlus Captions 빈]

## 10. 자주 묻는 질문 (FAQ)

### Q. 받아쓰기가 오래 걸립니다.

A. 1시간 영상 기준 평균 4~5분 소요됩니다. 오디오 트랙이 여러 개인 경우 추출
시간이 추가될 수 있습니다.

### Q. 화자가 잘못 분리되었습니다.

A. 체크박스 옆 ▼ 버튼으로 같은 화자를 모두 선택한 후, 상단에서 올바른 화자로
일괄 변경하세요. 화자가 부족하면 헤더의 👥 **화자 관리** 버튼에서 새 화자를
추가할 수 있습니다 (6. 화자 관리 참고).

### Q. 이전 작업을 이어서 하고 싶습니다.

A. 같은 시퀀스를 열면 상단에 "이미 받아쓴 기록이 있습니다" 배너가 표시됩니다.
**불러오기** 버튼을 클릭하세요.

### Q. 캡션을 다시 적용하고 싶습니다.

A. 자막편집 탭에서 수정 후 다시 **시퀀스에 자막 적용** 버튼을 누르면 됩니다.
기존 캡션 트랙은 직접 삭제 후 적용하는 것을 권장합니다.

### Q. 단축키가 동작하지 않습니다.

A. 패널 내 빈 공간을 한 번 클릭하여 포커스를 패널로 가져온 후 다시 시도해보세요.
한국어 입력 상태에서도 정상 동작합니다.
